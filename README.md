# 科研论文排版宏

一键格式化 Word 中英文混排科研论文正文的 VBA 宏。选中文本、运行宏，段落缩进 + 中文宋体 + 英文 Times New Roman + 字号行距 + 智能标点，全部一步到位。

## 功能

- **段落格式**：整段左缩进 + 段首额外缩进（默认各 2 字符），单倍行距，段前段后 0 磅
- **智能字体**：中文/CJK 自动设为宋体，英文/数字/希腊字母/科研符号自动设为 Times New Roman，统一小四（12 磅）
- **智能标点转换**：根据引号/括号内部内容的语言比例，自动将标点字符转换为对应中/英文形式并校正字体
- **性能优化**：关闭屏幕刷新，长文本排版不卡顿
- **安全可靠**：完整错误处理，异常时自动恢复屏幕刷新

## 快速开始

1. 打开你的 Word 文档，按 `Alt + F11` 进入 VBA 编辑器
2. 左侧「工程资源管理器」中右键你的文档 → **插入** → **模块**
3. 将 `科研论文排版宏.bas` 的全部内容粘贴到模块中，保存（建议另存为 `.docm` 格式）
4. 选中正文文本 → **视图** → **宏** → 运行「科研论文排版」

> **推荐**：给宏指定快捷键 `Ctrl + Alt + J`，之后选中文本一键排版。
>
> 路径：视图 → 宏 → 查看宏 → 选中「科研论文排版」→ 选项 → 指定快捷键。

## 个性化配置

代码顶部的常量区可以直接修改以适配不同学校的排版规范：

```vba
Private Const INDENT_LEFT As Integer = 2              ' 整段左缩进（字符数）
Private Const INDENT_FIRSTLINE As Integer = 2         ' 段首额外缩进（字符数）
Private Const FONT_CJK As String = "宋体"             ' 中文字体
Private Const FONT_LATIN As String = "Times New Roman" ' 西文字体
Private Const FONT_SIZE As Single = 12                ' 字号（磅），12 = 小四
```

## 常见问题

**Q：宏无法运行？**

依次打开 **文件 → 选项 → 信任中心 → 信任中心设置 → 宏设置**，选择「启用所有宏」或「禁用无数字签署的宏的所有宏（通知）」。

**Q：全局可用（所有文档都能用）怎么设置？**

将代码粘贴到 `Normal.dotm` 的模块中（VBA 编辑器左侧找到 Normal → 插入模块 → 粘贴代码）。

**Q：存档版 Sub 是什么？**

`科研论文排版_存档` 是不含智能标点功能的精简旧版，保留用于回溯参考。日常使用请运行 `科研论文排版`。
