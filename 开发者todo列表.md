## 🚀 P0 紧急/核心任务（v2.0.0 必须完成）

### 1. 增加完整的撤销支持
- **难度**：⭐
- **详细描述**：当前宏执行后无法通过一次撤销操作恢复，用户需要多次撤销才能回到排版前的状态，体验很差。
- **实现思路**：
  ```vba
  ' 在主程序开始时添加
  Application.UndoRecord.StartCustomRecord "科研论文排版"
  
  ' 在主程序结束时（ExitHandler中）添加
  Application.UndoRecord.EndCustomRecord
  ```
- **预期效果**：用户执行宏后，只需按一次`Ctrl+Z`即可完全恢复排版前的状态。
- **备注**：需要确保在所有退出路径（包括错误处理）中都能正确结束自定义撤销记录。

### 2. 扩展汉字识别范围至所有CJK统一汉字区
- **难度**：⭐
- **详细描述**：当前`GetChineseRatio`函数只识别基本汉字集（U+4E00-U+9FFF），无法识别扩展A、B、C、D、E、F区的生僻字。
- **实现思路**：
  ```vba
  ' 修改汉字判断条件
  If (charCode >= &H4E00 And charCode <= &H9FFF) Or _
     (charCode >= &H3400 And charCode <= &H4DBF) Or _ ' 扩展A区
     (charCode >= &H20000 And charCode <= &H2A6DF) Or _ ' 扩展B区
     (charCode >= &H2A700 And charCode <= &H2B73F) Or _ ' 扩展C区
     (charCode >= &H2B740 And charCode <= &H2B81F) Or _ ' 扩展D区
     (charCode >= &H2B820 And charCode <= &H2CEAF) Or _ ' 扩展E区
     (charCode >= &H2CEB0 And charCode <= &H2EBEF) Then ' 扩展F区
      chineseChars = chineseChars + 1
  End If
  ```
- **预期效果**：正确识别所有现代中文中可能出现的生僻字，提高中文占比计算的准确性。
- **备注**：扩展B区及以后的字符在旧版Word中可能显示异常，但不影响判断逻辑。

### 3. 完善破折号字体处理逻辑
- **难度**：⭐⭐
- **详细描述**：中文破折号（—）和英文em dash（—）是同一个Unicode字符（U+2014），当前代码将所有U+2014都设置为中文字体，在纯英文段落中会出现破折号字体不对的情况。
- **实现思路**：
  1. 遍历所有U+2014字符
  2. 检查该字符前后各3个字符的中文占比
  3. 如果前后中文占比≥0.5，设置为中文字体；否则设置为西文字体
- **预期效果**：根据上下文自动判断破折号应该使用的字体，解决纯英文段落中破折号字体错误的问题。
- **备注**：需要处理破折号在段落开头或结尾的边界情况。

## ✨ P1 重要/高价值任务（v2.0.0 主要亮点）

### 4. 增加更多标点符号的智能转换
- **难度**：⭐⭐
- **详细描述**：当前只处理了引号、括号、破折号和省略号，还需要添加以下标点的智能转换：
  - 逗号：`,` vs `，`
  - 句号：`.` vs `。`
  - 分号：`;` vs `；`
  - 冒号：`:` vs `：`
  - 感叹号：`!` vs `！`
  - 问号：`?` vs `？`
- **实现思路**：
  1. 创建一个新的子过程`ProcessCommonPunctuations`
  2. 遍历上述所有标点符号
  3. 对于每个标点，检查其前后各2个字符的中文占比
  4. 根据中文占比转换为对应的中英文标点并设置正确的字体
- **预期效果**：彻底解决中英文标点混用的问题，实现全文标点的自动统一。
- **备注**：
  - 需要特殊处理小数点（如`3.14`中的`.`不应该转换为`。`）
  - 需要特殊处理英文缩写中的句号（如`Mr.`中的`.`不应该转换为`。`）
  - 需要特殊处理时间格式中的冒号（如`12:30`中的`:`不应该转换为`：`）

### 5. 增加独立的"仅校正标点"功能
- **难度**：⭐
- **详细描述**：很多用户已经有了自己的段落和字体格式，只需要校正标点符号。
- **实现思路**：
  ```vba
  Sub 仅校正标点()
      On Error GoTo ErrorHandler
      
      If Selection.Type = wdNoSelection Then
          MsgBox "请先选中需要校正的文本！", vbExclamation, "提示"
          Exit Sub
      End If
      
      Application.ScreenUpdating = False
      Application.UndoRecord.StartCustomRecord "仅校正标点"
      
      Call FixQuoteFonts(Selection.Range)
      Call FixBracketFonts(Selection.Range)
      Call FixDashAndEllipsisFonts(Selection.Range)
      Call ProcessCommonPunctuations(Selection.Range) ' 待实现
      
  ExitHandler:
      Application.UndoRecord.EndCustomRecord
      Application.ScreenUpdating = True
      Exit Sub
      
  ErrorHandler:
      MsgBox "标点校正出错：" & Err.Description, vbCritical, "错误提示"
      Resume ExitHandler
  End Sub
  ```
- **预期效果**：用户可以单独使用标点校正功能，不修改已有的段落和字体格式。
- **备注**：需要添加对应的快捷键建议（如`Ctrl+Alt+K`）。

### 6. 优化长文档处理性能
- **难度**：⭐⭐
- **详细描述**：当前`GetChineseRatio`函数在处理非常长的文本（如几千字）时，由于逐字符扫描，可能会有轻微的性能下降。
- **实现思路**：
  1. 使用正则表达式一次性提取所有汉字和英文字母
  2. 统计匹配结果的数量，计算中文占比
  3. 保留原有的逐字符扫描作为备用方案（防止正则表达式在某些环境下不可用）
- **预期效果**：将长文本的中文占比计算速度提升5-10倍，进一步优化长文档排版体验。
- **备注**：需要测试正则表达式在不同Word版本中的兼容性。

## 🎨 P2 一般/锦上添花任务（v2.1.0 及以后）

### 7. 增加简单的进度条显示
- **难度**：⭐⭐
- **详细描述**：对于非常长的文档（几十页），用户不知道排版进度，可能会以为Word卡死了。
- **实现思路**：
  1. 创建一个用户窗体作为进度条
  2. 在主程序开始时显示进度条
  3. 在每个主要步骤（段落设置、字体设置、引号处理、括号处理等）更新进度
  4. 在主程序结束时关闭进度条
- **预期效果**：用户可以直观地看到排版进度，减少等待焦虑。
- **备注**：进度条应该是可选的，可以在常量区添加一个开关。

### 8. 增加"清除所有格式"功能
- **难度**：⭐
- **详细描述**：很多用户需要先清除文本的原有格式，再进行统一排版。
- **实现思路**：
  ```vba
  Sub 清除所有格式()
      On Error GoTo ErrorHandler
      
      If Selection.Type = wdNoSelection Then
          MsgBox "请先选中需要清除格式的文本！", vbExclamation, "提示"
          Exit Sub
      End If
      
      Application.ScreenUpdating = False
      Application.UndoRecord.StartCustomRecord "清除所有格式"
      
      Selection.ClearFormatting
      
  ExitHandler:
      Application.UndoRecord.EndCustomRecord
      Application.ScreenUpdating = True
      Exit Sub
      
  ErrorHandler:
      MsgBox "清除格式出错：" & Err.Description, vbCritical, "错误提示"
      Resume ExitHandler
  End Sub
  ```
- **预期效果**：一键清除选中文本的所有格式，恢复为默认样式。
- **备注**：需要添加警告提示，防止用户误操作。
